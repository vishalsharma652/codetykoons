# CodeTykoons

🚀 Official website of **CodeTykoons** — a full-service digital agency offering:

- 📈 Digital Marketing
- 🌐 Website Development
- 📱 Software App
- 🤖 AI Automation

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Vanilla CSS (CSS Modules)
- **Fonts**: Inter + Outfit (Google Fonts)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
